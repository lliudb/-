说明：
	插件结构说明：
		1、操作菜单，包括绘图按钮和操作按钮
		2、状态信息栏，显示当前操作状态
		3、绘图区，地图的编辑区域
		4、隐藏交互区域，编辑地图时显示的编辑操作区

操作状态机描述：
	SAVING状态在完成保存之后直接转换到IDLE状态
	EDITING状态下，提示保存/取消，然后回到IDLE状态
	绘制：
		IDLE => DRAWING => EDITING => SAVING
	选择并编辑
		IDLE => SELECTING => EDITING => SAVING
	中断:
		1、SELECTING => IDLE(主动取消状态)
		2、DRAWING => IDLE(前置SAVING操作)

对应状态可执行命令/界面：
	IDLE:
	DRAWING:
	SAVING:
	SELECTING:
	EDITING:


功能交互列表：
	添加地理原件操作：
		点状区域：残骸|村庄|要塞|祭坛|
		块状区域：山|湖|石林|森林|城市|
		线状区域：山脉|河|


	基础操作：
		系统操作：
			1、保存地图
			2、设置地图保存位置，甚至直接上传到服务器的指定位置（包括图像和数据）
			3、设置地图宽度X高度，设置刷新频率
			4、设置地图底色、地图其他基础信息
